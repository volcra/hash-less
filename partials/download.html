Get #Less

Prerequisites

Java 7 is required to build and run #Less.

Get the code

#Less is still in development, currently the only way to get it is to download the source from Github.

git clone git://github.com/volcra/hash-less.git

Or just download the ZIP and exract.

Install

Gradle is used as build tool. The Gradle wrapper is provided for quick development.

Windows

gradlew installApp

Unix

gradle installApp

This will create the executable with all the dependecies in directory ${basedir}\build\install\hash-less.

Verify

Execute the following command and you should expect the following output.

$>build\install\hash-less\bin\hash-less
usage: less [options]
Options
 -c,--compress       Compile the Less compressing the content.
 -h,--help           Print Help.
 -j,--join <FILE>    Before compiling, concatenate all scripts together in
                     the order they were
                     passed, and write them into the specified file.
                     Useful for building large projects.
 -o,--output <DIR>   Write out all compiled Less files into the specified
                     directory.
 -p,--print          Instead of writing out the Less as a file, print it
                     directly to stdout.

Now you are ready to head to the Usage page.
