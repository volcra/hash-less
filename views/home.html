#Less

The Groovy Less Compiler

Download GitHub Project

Download GitHub Project

#Less or Hash-Less is an implementation of the Less command line. It is written in Groovy and uses the core JavaScript compiler and Mozilla Rhino as libraries to compile the Less style sheet into CSS.

Many of the compiler options available in NodeJS are being ported to the Groovy version.

#Less can be integrated as a Command Line Tool, JAR dependency and Maven or Gradle Plugins.

Introduction

#Less is developed in Groovy. Why? Groovy is a dynamic language for the JVM. Comes with features that complement the Java development. Syntax is less verbose and Closures give the functional style missing in Java. These and some other features increase developer productivity by reducing boilerplate code when developing console applications.

Less is a dynamic stylesheet language. It extends CSS with dynamic behavior such as functions, operations, etc.

Here is an example of the equivalent code in Less and CSS.

Less 
@base: #f938ab;

.box-shadow(@style, @c) when (iscolor(@c)) {
  box-shadow:         @style @c;
  -webkit-box-shadow: @style @c;
  -moz-box-shadow:    @style @c;
}
.box-shadow(@style, @alpha: 50%) when (isnumber(@alpha)) {
  .box-shadow(@style, rgba(0, 0, 0, @alpha));
}
.box {
  color: saturate(@base, 5%);
  border-color: lighten(@base, 30%);
  div { .box-shadow(0 0 5px, 30%) }
}
CSS 
.box {
  color: #fe33ac;
  border-color: #fdcdea;
}
.box div {
  box-shadow: 0 0 5px rgba(0, 0, 0, 0.3);
  -webkit-box-shadow: 0 0 5px rgba(0, 0, 0, 0.3);
  -moz-box-shadow: 0 0 5px rgba(0, 0, 0, 0.3);
}

Quick Start

Command-Line

Download and build from source. For instructions go to Download.

Then compile your scripts.

$>build\install\hash-less\bin\hash-less -c app.less

JAR dependency Under development

API

Maven

<dependency>
  <groupId>org.volcra.less</groupId>
  <artifactId>less-compiler</artifactId>
  <version>1.0.0-SNAPSHOT</version>
</dependency>

Ivy

<dependency org="org.volcra.less" name="less-compiler" rev="1.0.0-SNAPSHOT" />

Gradle/Grails

compile 'org.volcra.less:less-compiler:1.0.0-SNAPSHOT'

Groovy Grape

@Grapes(
  @Grab(group='org.volcra.less', module='less-compiler', version='1.0.0-SNAPSHOT')
