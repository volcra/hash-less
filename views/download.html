Get #Less

Prerequisites

Java 7 is required to build and run #Less.

Get the code

#Less is in development and it may change continuosly until it reaches a GA version.

Currently the only way to get it downloading the source from GitHub.

git clone git://github.com/volcra/hash-less.git

Or just download the ZIP and exract in a directory of your choice.

Install

Gradle is used as build tool. The Gradle wrapper is provided for quick development. Gradle will be automatically downloaded and used to run the build.

Windows

In Windows the batch script is called gradlew.bat.

gradlew installApp

Unix

In Unix the batch script is gradlew.

gradle installApp

The installApp task is provided by the application plugin. This will create the executable with all the dependencies in the directory ${basedir}\build\install\hash-less.

Verify

Execute the command and you should expect the following output.

$>build\install\hash-less\bin\hash-less
usage: less [options]
Options
-c,--compress       Compile the Less compressing the content.
-h,--help           Print Help.
-j,--join <FILE>    Before compiling, concatenate all scripts together in
                    the order they were
                    passed, and write them into the specified file.
                    Useful for building large projects.
-o,--output <DIR>   Write out all compiled Less files into the specified
                    directory.
-p,--print          Instead of writing out the Less as a file, print it
                    directly to stdout.

Now you are ready to head to the Usage page.
