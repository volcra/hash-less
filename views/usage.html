Usage

How to get started

Getting Started

You need first to get #Less, go to the Download page for more information on how to install and set up #Less.

Getting Help

Run #Less without any options or passing -h or --help to get help.

$>build\install\hash-less\bin\hash-less
usage: less [options]
Options
-c,--compress       Compile the Less compressing the content.
-h,--help           Print Help.
-j,--join <FILE>    Before compiling, concatenate all scripts together in
                    the order they were
                    passed, and write them into the specified file.
                    Useful for building large projects.
-o,--output <DIR>   Write out all compiled Less files into the specified
                    directory.
-p,--print          Instead of writing out the Less as a file, print it
                    directly to stdout.

Command-Line Usage

Options

-c,--compress

Compress will minify the compiled CSS.

-j,--join <FILE>

This is usefull for projects with multiple sources. Before compiling, the list of files will be joined togheter in the order they were passed.

You may also use @import in your LESS style sheets to produce same effect.

-o,--output <DIR>

By default compiled CSS are written to the same location fo the LESS style sheet. To change that behavior use this option.

-p,--print

Write the compiled CSS to stdout.

You may send the output to a file of your choice.

$>build\install\hash-less\bin\hash-less -p src\test\resources\test.less > test.css

Sample File

This file is located at ${basedir}\src\test\resources\test.less. Where basedir is the path where you unpacked the application.

@base: #f938ab;

.box-shadow(@style, @c) when (iscolor(@c)) {
  box-shadow:         @style @c;
  -webkit-box-shadow: @style @c;
  -moz-box-shadow:    @style @c;
}
.box-shadow(@style, @alpha: 50%) when (isnumber(@alpha)) {
  .box-shadow(@style, rgba(0, 0, 0, @alpha));
}
.box {
  color: saturate(@base, 5%);
  border-color: lighten(@base, 30%);
  div { .box-shadow(0 0 5px, 30%) }
}

Then execute the hash-less command. The option -p will instruct the command to print the result to the console.

This will compile the source file test.less and produce the following output.

$>build\install\hash-less\bin\hash-less -p src\test\resources\test.less
.box {
  color: #fe33ac;
  border-color: #fdcdea;
}
.box div {
  box-shadow: 0 0 5px rgba(0, 0, 0, 0.3);
  -webkit-box-shadow: 0 0 5px rgba(0, 0, 0, 0.3);
  -moz-box-shadow: 0 0 5px rgba(0, 0, 0, 0.3);
}
